How to run (assuming you have node installed):
1. Extarct the zip file.
2. Go to the project folder and run the command: "npm start-smee". This runs the smee tool.
3. Go the project folder and run the command: "npm start". This runs the app.
4. Register the 3 relevant webhooks on a random organization to the following url: https://smee.io/dIYRLywTcJKdDx7.
5. Try to perform the actions. If the action is suspicious, you will get a message in the log.
